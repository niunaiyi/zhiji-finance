# Report 模块规范

**所属 Agent：** Agent 2
**职责：** 三大财务报表生成

## 报表取数逻辑
所有报表从 `balance` 和 `balance_aux` 取数，不重新汇总凭证行。

## 资产负债表
- 时点数据（某期末余额）
- 资产 = 负债 + 所有者权益（必须平衡，否则报错）
- 按 `account.element_type` 分类汇总

## 利润表
- 期间数据（某期间发生额）
- 收入 - 成本 - 费用 = 利润

## 现金流量表
- 间接法：从净利润调整
- 需要预先配置现金流量项目与科目的映射关系

## 报表格式
- 支持按期间查询（单期/累计）
- 支持同比/环比数据
- 导出格式：JSON（前端渲染）/ Excel
