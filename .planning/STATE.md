# 项目当前状态 (STATE)

## 当前概况
- **项目名称**: 智积财务管理系统 (Zhiji Finance)
- **核心框架**: Apiato v13 (Laravel 11) + React (Vite)
- **数据库**: PostgreSQL 15
- **功能完备性**: 基础科目、凭证核心、存货、固定资产、薪酬、财务报表已完成。

## 技术基准
- **租户隔离**: 严格使用 `X-Company-Id` 请求头，Middleware 自动注入。
- **数据一致性**: 凭证过账触发 `VoucherPostedEvent` 更新总账与辅助账余额。
- **报表逻辑**: 基于 `balances` 和 `balance_aux` 实时的汇总聚合。

## 待解决问题 (Backlog)
- 业务单据（存货出库、资产折旧、工资发放）与总账凭证的自动化对接。
- 完整的 RBAC 权限体系与账套管理的深度安全性验证。
- 更多财务分析功能（如账龄分析的 UI 展示优化）。
